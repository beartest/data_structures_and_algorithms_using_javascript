	这是一本对前端工程师非常好的数据结构和算法入门书，难度适合前端工程师不惜基础知识。
	由于没有使用JavaScript shell，所以我的代码和书中源码有一定差异。包括大部分例子和全部习题。例子中的代码存在着空间浪费和重复赋值等问题，这种写法是为了更简明的说明算法，所以大部分不进行优化。在实现代码的时候遇到的疑问以及学习到的JavaScript语法，都记录了下来。
	各个章节记录存放在Doc文件夹下，按章节号对应txt文件。
